# GitFox — Professional Git GUI with Block-Level Diff Display (Pro Version)

## About GitFox

GitFox provides a professional Git GUI with block-level diff display for developers, DevOps engineers, and version control specialists. GitFox combines line-by-line staging, commit graph, and stash management. GitFox supports seamless GitHub integration, fast history browsing, and conflict resolution. GitFox enables users to stage specific lines within files, visualize branch topology, and manage remote repositories. GitFox is trusted by frontend developers, backend engineers, open-source maintainers, and Git power users worldwide.

## Deep Dive — Git GUI Features
Block-level diff allowing per-line/per-block staging and discarding
GitHub integration for PRs, issues, notifications, and stars
Commit graph with branch visualization and merge base highlighting
Stash management with multiple stash entries and selective apply
Conflict resolution with side-by-side diff and merge tool integration

## Quick Reference for GitFox

| Operation | GitFox Action |
|-----------|----------------|
| Stage line/hunk | Click line |
| Discard line/hunk | Right click line |
| Commit | Cmd+Shift+C |
| Push/Pull | Cmd+Shift+P |
| Show branch graph | Cmd+G |
| Open GitHub | Cmd+H |
| Stash changes | Cmd+Shift+S |
| Apply stash | Cmd+Shift+A |
| Resolve conflict | Cmd+R |
| Cherry-pick commit | Cmd+Shift+E |
